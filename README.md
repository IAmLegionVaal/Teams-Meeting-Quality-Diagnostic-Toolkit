# Teams Meeting Quality Diagnostic Toolkit

A public-safe PowerShell toolkit for Teams meeting quality troubleshooting preparation.

## Features

- Microsoft Teams endpoint connectivity checks
- Audio/video device inventory
- Network adapter context
- Teams process context
- Meeting quality checklist output
- CSV, JSON, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Teams_Meeting_Quality_Diagnostic_Toolkit.ps1
```

## Safety

Diagnostic-only. It does not collect meeting content, chat, recordings, or user files.
